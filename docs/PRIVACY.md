## IMPORTANT: DEMO PLATFORM PRIVACY NOTICE

**THIS IS A DEMONSTRATION PLATFORM. NO REAL PERSONAL DATA OR FINANCIAL INFORMATION IS STORED OR PROCESSED.**

TEOS BankChain is currently operating in **DEMO MODE** for testing and evaluation. This Privacy Policy explains our data practices for the demo platform.

### Key Privacy Points for Demo Mode

1. **No Real Personal Data**: We do not collect or store real personal financial information in demo mode.

2. **Demo Credentials Only**: Email and password fields are for demonstration accounts only and are not treated as sensitive data.

3. **No Payment Information**: No real payment information, credit card numbers, or bank account details are collected.

4. **Testing Data Only**: All data entered is for testing purposes and may be visible to TEOS Egypt staff and partners.

5. **No Data Protection Guarantees**: Demo data is not subject to the same protection standards as production data would be.

### 1.1 Demo Account Information

### 1.2 Usage Information

- **Demo Transactions**: Simulated transaction history

### 1.4 What We DO NOT Collect
- ✗ Real personal identification documents
- ✗ Real financial account information
- ✗ Real payment card details
- ✗ Government-issued ID numbers
- ✗ Real transaction data
- ✗ Sensitive personal information
